HTML الدرس 1

أساسيات تطوير الويب

HTML، CSS، و JavaScript هي اللغات الأساسية لتطوير مواقع الويب.

ما هو HTML؟

HTML (HyperText Markup Language) تُستخدم لإنشاء هيكل الصفحة وتحديد المحتوى، مثل العناوين، الفقرات، والصور.

ما هو CSS؟

CSS (Cascading Style Sheets) تتحكم في تصميم الصفحة، مثل الألوان، الخطوط، والتنسيقات لجعل الموقع أكثر جاذبية.

ما هو JavaScript؟

JavaScript تضيف التفاعلية إلى الموقع، مثل النقرات، الحركات، والتحديثات الديناميكية دون إعادة تحميل الصفحة.

عند استخدامها معًا، تتيح لنا بناء مواقع ويب متكاملة وسريعة الاستجابة تجمع بين المحتوى، التصميم، والتفاعل.

HTML الدرس 6

في هاذا الدرس سوف نتحدث عن ادخال البيانات من قبل المستخدم



شرح الوسم 



<input type="text" name="name" placeholder="أدخل اسمك">



هاذا هوه وسم ادخال البيانات من قبل المستخدم اسمه ان بوت الوسم فردي اي يتكون من فتح فقط ولايوجد اغلاق للوسم 
الان سوف نقوم بتشريح الوسم 



1. input

هو الوسم الأساسي الذي يُستخدم لإنشاء عنصر إدخال في HTML.

يسمح للمستخدم بإدخال البيانات في النموذج، مثل النصوص أو الأرقام أو الخيارات.


2. type="text"

type: هذه السمة تحدد نوع الإدخال الذي سيظهر للمستخدم.

في هذه الحالة، النوع هو text، مما يعني أن الحقل سيكون عبارة عن مربع نص يمكن للمستخدم كتابة نص فيه.

أنواع أخرى يمكن أن تستخدم هي: password (إخفاء النصوص المدخلة)، email (للتحقق من صيغة البريد الإلكتروني)، number (لإدخال الأرقام)، وغيرها.


3. name="name"

name: هذه السمة تُستخدم لتسمية حقل الإدخال.

عندما يتم إرسال النموذج إلى الخادم، سيتم إرسال القيمة المدخلة في هذا الحقل مع الاسم المحدد هنا (في هذه الحالة، "name").

يُستخدم هذا الاسم لتحديد البيانات التي أُرسلت من النموذج على الخادم.


4. placeholder="أدخل اسمك"

placeholder: هذه السمة تُعرض نصًا داخل الحقل عندما يكون فارغًا، وهو موجه للمستخدم ليعرف ماذا يجب عليه إدخال.

في هذه الحالة، النص المعروض هو "أدخل اسمك"، وهو تلميح للمستخدم بضرورة إدخال اسمه في الحقل.

عندما يبدأ المستخدم في الكتابة، يختفي النص المعروض في الـ placeholder.


مثال:

عند استخدام هذا الوسم في نموذج، سيظهر للمستخدم حقل نصي يحتوي على النص "أدخل اسمك" كدليل لما يجب أن يكتبه. عندما يبدأ المستخدم الكتابة، سيختفي هذا النص، ويبدأ في إدخال بياناته.





شرح جميع انوع type



  
  هنا جمبع انوع input

1. type="text": يستخدم لإنشاء حقل نصي يسمح للمستخدم بإدخال نص.


2. type="password": مشابه لـ "text"، ولكن النص المدخل يتم إخفاؤه (مثل كلمة المرور).


3. type="radio": يتيح للمستخدم اختيار خيار واحد فقط من مجموعة من الخيارات.


4. type="checkbox": يتيح للمستخدم اختيار أو إلغاء اختيار خيارات متعددة.


5. type="submit": ينشئ زرًا لإرسال النموذج.


6. type="reset": ينشئ زرًا لإعادة تعيين جميع الحقول في النموذج إلى قيمها الافتراضية.


7. type="button": ينشئ زرًا يمكن تخصيصه بأي وظيفة باستخدام JavaScript.


8. type="file": يتيح للمستخدم اختيار ملف من جهاز الكمبيوتر لتحميله.


9. type="hidden": ينشئ حقلًا مخفيًا لا يظهر للمستخدم.


10. type="image": ينشئ زرًا يحتوي على صورة يمكن النقر عليها لإرسال النموذج.


11. type="date": يسمح للمستخدم باختيار تاريخ باستخدام واجهة محددة.


12. type="email": يسمح للمستخدم بإدخال عنوان بريد إلكتروني. يقوم المتصفح بالتحقق من صحة التنسيق.


13. type="number": يسمح للمستخدم بإدخال رقم فقط، ويمكن تحديد القيم العليا والدنيا.


14. type="tel": يسمح للمستخدم بإدخال رقم هاتف.


15. type="url": يسمح للمستخدم بإدخال عنوان URL. يقوم المتصفح بالتحقق من صحة التنسيق.


16. type="range": يتيح للمستخدم تحديد قيمة ضمن نطاق باستخدام شريط تمرير.


17. type="color": يسمح للمستخدم باختيار لون باستخدام واجهة اختيار الألوان.


18. type="datetime-local": يسمح للمستخدم باختيار تاريخ ووقت معًا.


19. type="month": يسمح للمستخدم باختيار شهر وسنة فقط.


20. type="week": يسمح للمستخدم باختيار أسبوع في السنة.


21. type="time": يسمح للمستخدم باختيار وقت فقط (بدون تاريخ).


22. type="search": يتيح للمستخدم إدخال نص بحث. قد يعرض بعض المتصفحات واجهة خاصة للبحث.


23. type="image": يمكن للمستخدم إدخال صورة باستخدام مربع حوار لاختيار صورة.






شرح سمات ال input




إليك شرح جميع سمات الوسم  inputفي HTML:

1. type: يحدد نوع الحقل (مثل text, password, checkbox, radio, إلخ).


2. name: يحدد اسم الحقل ويستخدم لإرسال البيانات إلى الخادم (خاصة في نماذج HTML).


3. value: يحدد القيمة الافتراضية للحقل أو القيمة التي يتم إرسالها إلى الخادم عند إرسال النموذج.


4. id: يحدد معرفًا فريدًا للحقل يمكن استخدامه للاستهداف باستخدام JavaScript أو CSS.


5. placeholder: يحدد النص الذي يظهر داخل الحقل عندما يكون فارغًا، ويختفي عندما يبدأ المستخدم في الكتابة.


6. required: إذا كانت هذه السمة موجودة، فإن الحقل يصبح إلزاميًا ويجب على المستخدم تعبئته قبل إرسال النموذج.


7. disabled: إذا كانت هذه السمة موجودة، يتم تعطيل الحقل ولا يمكن التفاعل معه (لا يمكن للمستخدم تعبئته أو اختياره).


8. readonly: يجعل الحقل غير قابل للتعديل (يمكن التفاعل معه ولكن لا يمكن تغيير قيمته).


9. maxlength: يحدد الحد الأقصى لعدد الأحرف التي يمكن للمستخدم إدخالها في الحقل النصي.


10. size: يحدد عدد الأحرف المرئيين في الحقل النصي (يؤثر على عرض الحقل).


11. min: يحدد الحد الأدنى للقيمة التي يمكن إدخالها في الحقل (يُستخدم عادة مع أنواع مثل number, range, date).


12. max: يحدد الحد الأقصى للقيمة التي يمكن إدخالها في الحقل (يُستخدم عادة مع أنواع مثل number, range, date).


13. step: يحدد مقدار القيم التي يمكن للمستخدم زيادتها أو تقليلها عند إدخال قيم عددية (يُستخدم عادة مع number و range).


14. autofocus: إذا كانت هذه السمة موجودة، يتم تحديد التركيز تلقائيًا على الحقل عند تحميل الصفحة.


15. autocorrect: يستخدم لتحديد ما إذا كان يجب تمكين التصحيح التلقائي للنصوص المدخلة (يُستخدم عادة في الحقول التي تحتوي على نصوص).


16. autocomplete: يحدد ما إذا كان يجب تمكين إكمال تلقائي للقيم المدخلة في الحقل (يسمح للمستخدمين بالاختيار من القيم السابقة).


17. form: يحدد النموذج الذي ينتمي إليه الحقل (يُستخدم عندما يكون الحقل خارج النموذج ولكن ترغب في ربطه به).


18. formaction: يحدد عنوان URL الذي يتم إرسال النموذج إليه عند إرسال النموذج باستخدام هذا الحقل.


19. formenctype: يحدد كيفية ترميز البيانات عند إرسال النموذج باستخدام هذا الحقل (يُستخدم مع formaction).


20. formmethod: يحدد طريقة إرسال البيانات عند إرسال النموذج باستخدام هذا الحقل (GET أو POST).


21. formtarget: يحدد نافذة أو إطار الهدف الذي سيتم إرسال البيانات إليه.


22. formnovalidate: إذا كانت هذه السمة موجودة، فإنه يتم تعطيل التحقق من صحة البيانات للنموذج عند إرسال البيانات باستخدام هذا الحقل.


23. multiple: يسمح للمستخدم بتحديد أكثر من ملف لتحميله (يُستخدم مع file).


24. pattern: يحدد تعبيرًا عاديًا (Regular Expression) للتحقق من صحة المدخلات (يُستخدم مع أنواع مثل text, email, tel).


25. spellcheck: يحدد ما إذا كان يجب تمكين التدقيق الإملائي في الحقل.


26. inputmode: يحدد نوع لوحة المفاتيح التي يجب أن تظهر على الأجهزة المحمولة (مثل numeric, tel, email).


27. capture: يتيح للمستخدم التقاط صورة أو فيديو باستخدام كاميرا الجهاز أو الميكروفون.

CSSالدرس 2

في هذا الدرس سنشرح الخصائص المتعلقة في النصوص

1. تحديد نوع الخط (font-family) 

p {
  font-family: Arial, sans-serif; /* استخدام خط Arial */
}

p {
  font-family: 'Times New Roman', serif; /* استخدام خط Times New Roman */
}

p {
  font-family: 'Courier New', monospace; /* استخدام خط Courier New */
}

تستخدم font-family لتحديد نوع الخط المستخدم في النص. يمكن استخدام أسماء الخطوط المعروفة مثل Arial أو Times New Roman أو خطوط خاصة عبر قيم متعددة.

2. تحديد حجم الخط (font-size) 

p {
  font-size: 16px; /* تحديد حجم الخط 16 بكسل */
}

p {
  font-size: 1.5em; /* تحديد حجم الخط بالنسبة لحجم الخط الأب */
}

p {
  font-size: 20pt; /* تحديد حجم الخط باستخدام نقاط (pt) */
}

تستخدم font-size لتحديد حجم الخط. يمكن تحديد الحجم بوحدات مثل البكسل (px)، أو النسبة المئوية (em)، أو النقاط (pt).

3. التحكم في سمك الخط (font-weight) 

p {
  font-weight: normal; /* تحديد سمك الخط الطبيعي */
}

p {
  font-weight: bold; /* تحديد الخط العريض */
}

p {
  font-weight: 700; /* تحديد سمك الخط باستخدام الأرقام */
}

تستخدم font-weight لتحديد سمك الخط. يمكن أن تكون القيم مثل normal أو bold، أو قيمة عددية مثل 700 لزيادة سمك الخط.

4. التحكم في ميلان الخط (font-style) 

p {
  font-style: normal; /* تحديد نمط الخط العادي */
}

p {
  font-style: italic; /* تحديد نمط الخط المائل */
}

p {
  font-style: oblique; /* تحديد نمط الخط المائل بدرجة صغيرة */
}

تستخدم font-style لتحديد نمط الخط. يمكن أن تكون القيم مثل normal أو italic أو oblique لخط مائل بدرجة خفيفة.

5. التحكم في محاذاة النص (text-align) 

p {
  text-align: left; /* محاذاة النص إلى اليسار */
}

p {
  text-align: center; /* محاذاة النص إلى المنتصف */
}

CSS الدرس 3

في هاذا الدرس سوف نشرح جميع خصائص الصور

1. تحديد عرض الصورة (width) 

img {
  width: 100%; /* عرض الصورة يتناسب مع عرض العنصر الأب */
}

img {
  width: 300px; /* تعيين عرض الصورة بـ 300 بكسل */
}

تُستخدم خاصية width لتحديد عرض الصورة. يمكن تعيين العرض باستخدام وحدات مثل px و %.

2. تحديد ارتفاع الصورة (height) 

img {
  height: 200px; /* تعيين ارتفاع الصورة بـ 200 بكسل */
}

img {
  height: auto; /* يسمح للصورة بالحفاظ على نسبها الأصلية */
}

تُستخدم خاصية height لتحديد ارتفاع الصورة. يمكن تحديدها باستخدام وحدات مثل px أو تركها auto للحفاظ على النسب.

3. تغطية الصورة (object-fit) 

img {
  object-fit: cover; /* الصورة ستغطي العنصر بأكمله مع الحفاظ على النسب */
}

img {
  object-fit: contain; /* الصورة ستتسع داخل العنصر مع الحفاظ على النسب */
}

تُستخدم خاصية object-fit للتحكم في كيفية ملاءمة الصورة داخل العنصر. cover يعني أن الصورة ستغطي كامل المساحة، بينما contain يعني أن الصورة ستظل كاملة ولكن قد تحتوي على مساحات فارغة.

4. تحديد صورة خلفية باستخدام صورة (background-image) 

div {
  background-image: url('image.jpg');
}

تُستخدم خاصية background-image لتعيين صورة خلفية للعنصر. يمكن استخدام روابط أو مسارات محلية للصور.

5. تحديد حجم الصورة التلقائي (auto) 

img {
  width: auto; /* السماح للصورة بالحفاظ على حجمها الأصلي */
  height: auto; /* السماح للصورة بالحفاظ على ارتفاعها الأصلي */
}

تُستخدم auto في عرض وارتفاع الصورة لتوفير الحجم الأصلي للصورة دون تعديل.

6. تحديد حدود الصورة (border) 

img {
  border: 2px solid black; /* تعيين حدود للصورة بسمك 2 بكسل ولون أسود */
}

تُستخدم خاصية border لتحديد سمك ولون الحدود حول الصورة.

7. إضافة ظل للصورة (box-shadow) 

img {
  box-shadow: 10px 10px 5px grey; /* إضافة ظل للصورة */
}

تُستخدم خاصية box-shadow لإضافة تأثير الظل حول الصورة. يمكن تخصيص حجم واتجاه الظل.

8. تقليص الصورة لتناسب العنصر (max-width, max-height) 

img {
  max-width: 100%; /* تعيين الحد الأقصى لعرض الصورة */
  max-height: 300px; /* تعيين الحد الأقصى لارتفاع الصورة */
}

تُستخدم خاصية max-width و max-height لتحديد أقصى عرض وارتفاع يمكن أن تصل إليه الصورة.

9. جعل الصورة دائرية (border-radius) 

img {
  border-radius: 50%; /* جعل الصورة دائرية */
}

تُستخدم border-radius لتحديد حواف مستديرة للصورة. عند تعيينها بـ 50%، تتحول الصورة إلى دائرة إذا كانت مربعة.

10. ضبط صورة على الطفو (float) 

img {
  float: left; /* تعويم الصورة إلى اليسار */
}

img {
  float: right; /* تعويم الصورة إلى اليمين */
}

تُستخدم خاصية float لتحريك الصورة إلى اليسار أو اليمين داخل العنصر، مما يسمح بتدفق النص حول الصورة.

11. تحديد مدى شفافية الصورة (opacity) 

img {
  opacity: 0.5; /* تعيين الشفافية بنسبة 50% */
}

تُستخدم خاصية opacity لتحديد مدى شفافية الصورة. القيم تتراوح بين 0 (شفاف بالكامل) و 1 (غير شفاف).

12. تعديل صورة باستخدام تأثيرات الفلتر (filter)

}

  filter: blur(5px); /* إضافة تأثير تشويش على الصورة */
}

تُستخدم خاصية filter لإضافة تأثيرات على الصورة مثل التدرج الرمادي أو التشويش أو السطوع. يمكن تخصيص هذه التأثيرات حسب الحاجة.

13. منع الصورة من التمدد (object-position)

}

img {
  object-position: top left; /* تعيين الصورة في الزاوية العلوية اليسرى */
}

تُستخدم خاصية object-position لتحديد موضع الصورة داخل العنصر عندما يتم تغيير حجمها باستخدام object-fit.

14. منع تكبير الصورة أو تصغيرها عند تغيير حجم العنصر (image-rendering)

  image-rendering: pixelated; /* عرض الصورة بطريقة بيكسيلية */
}

تُستخدم خاصية image-rendering لتحديد كيف يجب عرض الصورة عند تغيير حجمها. القيمة pixelated تجعل الصورة تظهر بشكل بيكسلي.

15. جعل الصورة تتوسع لتغطي المساحة المتاحة (object-fit: fill)

  object-fit: fill; /* الصورة ستتوسع لتغطي العنصر بالكامل */

تُستخدم object-fit: fill لتمديد الصورة لتغطية المساحة الكاملة للعنصر، مما قد يؤدي إلى تشويه الصورة.

16. خاصية `image-rendering`

}

}

img {
  image-rendering: pixelated; /* يظهر الصورة بشكل بيكسلي */
}

تُستخدم خاصية image-rendering لتحديد كيفية عرض الصورة عند تكبيرها أو تصغيرها. يمكن استخدامها لعرض الصورة بشكل طبيعي أو بشكل بيكسلي.

17. خاصية `mask-image` 

img {
  mask-image: url('mask.png'); /* تعيين صورة كقناع */
}

تُستخدم خاصية mask-image لإضافة قناع للصورة، حيث يتم إخفاء بعض الأجزاء من الصورة بناءً على الصورة القناع المحددة.

18. خاصية `mask-size` 

img {

تُستخدم خاصية box-shadow لإضافة تأثير الظل للزر. يمكن تخصيص المسافة، الحجم، واللون.

17. تأثير التمويه عند التمرير (Transition) 

button {
  transition: background-color 0.3s ease; /* انتقال تأثير تغيير اللون */
}

تُستخدم خاصية transition لتحديد تأثيرات التغيير التدريجي للخصائص عند التفاعل مع الزر مثل تغيير اللون أو الحجم.

18. الزر الشفاف (Opacity) 

button {
  opacity: 0.8; /* جعل الزر شفافًا بنسبة 80% */
}

تُستخدم خاصية opacity لتحديد شفافية الزر. القيمة 1 تعني عدم وجود شفافية، بينما القيمة 0 تعني الشفافية التامة.

19. تأثير التمويه عند التركيز (Focus) 

button:focus {
  background-color: #45a049; /* تغيير الخلفية عند التركيز */
}

تُستخدم خاصية :focus لتحديد تأثيرات الزر عندما يحصل على التركيز (عند النقر عليه أو تحديده باستخدام التاب).

20. إزالة تأثير الظل عند الضغط (Active) 

button:active {
  box-shadow: none; /* إزالة الظل عند الضغط على الزر */
}

تُستخدم خاصية :active لتحديد التأثيرات التي تحدث عند الضغط على الزر، مثل إزالة الظلال أو تغيير اللون.

21. تأثير النص المائل (Font Style) 

button {
  font-style: italic; /* جعل النص مائلًا */
}

تُستخدم خاصية font-style لتحديد نمط النص داخل الزر. يمكن تعيينه كـ `normal`, `italic`, أو `oblique`.

22. إضافة تأثيرات الخط العلوي (Text Transform) 

button {
  text-transform: uppercase; /* تحويل النص إلى أحرف كبيرة */
}

تُستخدم خاصية text-transform لتحديد كيفية تحويل النص داخل الزر. القيم الممكنة هي `uppercase`, `lowercase`, و `capitalize`.

23. تحديد الخلفية المتدرجة (Background Gradient) 

button {
  background: linear-gradient(to right, red, yellow); /* تدرج لوني من الأحمر إلى الأصفر */
}

تُستخدم خاصية background مع التدرجات اللونية (مثل linear-gradient) لإنشاء خلفيات متدرجة للزر بدلاً من اللون الثابت.

24. حجم الأيقونة داخل الزر (Icon Size) 

button i {
  font-size: 20px; /* تغيير حجم الأيقونة داخل الزر */
}

تُستخدم هذه الخاصية لضبط حجم الأيقونات داخل الأزرار عند استخدام مكتبات الأيقونات مثل Font Awesome.

25. التأثير عند التمرير على النص (Text Shadow) 

button {
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3); /* إضافة ظل للنص داخل الزر */
}

تُستخدم خاصية text-shadow لإضافة ظل للنص داخل الزر. يمكن تخصيص المسافة، اللون، والحجم.

26. الحد الأدنى للطول (Min Height) 

button {
  min-height: 40px; /* تحديد الحد الأدنى للطول */
}

تُستخدم خاصية min-height لتحديد الحد الأدنى للطول الذي يجب أن يكون عليه الزر، حتى لو كان المحتوى أقل من ذلك.

27. الحد الأقصى للطول (Max Height) 

button {
  max-height: 60px; /* تحديد الحد الأقصى للطول */
}

تُستخدم خاصية max-height لتحديد الحد الأقصى للطول الذي يمكن أن يصل إليه الزر.

28. الحد الأدنى للعرض (Min Width) 

button {
  min-width: 100px; /* تحديد الحد الأدنى للعرض */
}

تُستخدم خاصية min-width لتحديد الحد الأدنى لعرض الزر.

29. الحد الأقصى للعرض (Max Width) 

button {
  max-width: 200px; /* تحديد الحد الأقصى للعرض */
}

تُستخدم خاصية max-width لتحديد الحد الأقصى للعرض الذي يمكن أن يصل إليه الزر.

30. تحديد محاذاة النص (Text Align) 

button {
  text-align: center; /* محاذاة النص في المنتصف */
}

تُستخدم خاصية text-align لمحاذاة النص داخل الزر. القيم الممكنة هي `left`, `center`, `right`.

31. الشفافية عند الضغط (Active Opacity) 

button:active {
  opacity: 0.7; /* جعل الزر شفافًا عند الضغط */
}

تُستخدم خاصية opacity عند الضغط على الزر لجعل الزر يصبح أكثر شفافية.

32. تأثير التحويل عند التمرير (Hover Transform) 

button:hover {
  transform: scale(1.1); /* تكبير الزر عند التمرير */
}

تُستخدم خاصية transform مع :hover لإضافة تأثيرات مثل التكبير أو التدوير عند التمرير فوق الزر.

CSS الدرس 6

في هذا الدرس سنتعرف على خصائص CSS المختلفة لتنسيق الوسم div في HTML.

1. تحديد العرض والارتفاع (Width & Height) 

div {
  width: 200px;  /* العرض 200 بكسل */
  height: 100px; /* الارتفاع 100 بكسل */
}

div {
  width: 50%;  /* العرض نصف عرض العنصر الأب */
  height: 50vh; /* الارتفاع نصف ارتفاع الشاشة */
}

تحدد width و height عرض وارتفاع العنصر. يمكن استخدام وحدات مثل البكسل (px)، النسبة المئوية (%)، أو وحدات العرض والارتفاع مثل (vw, vh).

2. تحديد الخلفية (Background) 

div {
  background-color: red; /* لون خلفية أحمر */
  background-image: url('background.jpg'); /* إضافة صورة خلفية */
  background-size: cover; /* جعل الصورة تغطي العنصر بالكامل */
}

تستخدم خاصية background لتحديد لون أو صورة الخلفية للعنصر.

3. التحكم في التباعد الداخلي (Padding) 

div {
  padding: 10px; /* مسافة داخلية متساوية من جميع الجهات */
}

div {
  padding: 10px 20px; /* 10px أعلى وأسفل، 20px يمين ويسار */
}

div {
  padding: 10px 15px 20px 25px; /* ترتيب: أعلى - يمين - أسفل - يسار */
}

تحدد padding المسافة بين محتوى العنصر وحدوده. يمكن تحديد المسافة نفسها لكل الجوانب أو تخصيصها لكل جهة على حدة.

4. التحكم في التباعد الخارجي (Margin) 

div {
  margin: 20px; /* مسافة خارجية متساوية من جميع الجهات */
}

div {
  margin: 10px 15px; /* 10px أعلى وأسفل، 15px يمين ويسار */
}

div {
  margin: 10px 15px 20px 25px; /* ترتيب: أعلى - يمين - أسفل - يسار */
}

تحدد margin المسافة بين العنصر والعناصر الأخرى المحيطة به. يمكن تخصيص المسافة لكل جهة على حدة.

5. تحديد الحواف (Border) 

div {
  border: 2px solid black; /* حافة بسمك 2 بكسل ولون أسود */
  border-radius: 10px; /* جعل الحواف مدورة */
}

تستخدم border لتحديد الحافة الخارجية للعنصر. يمكن تخصيص سمك الحافة، نوعها (مثل solid) ولونها. أما border-radius فيستخدم لجعل الحواف مدورة.

6. التحكم في إخفاء المحتوى (Visibility) 

div {
  visibility: hidden; /* إخفاء العنصر مع الحفاظ على المساحة */
}

تستخدم visibility لتحديد ما إذا كان العنصر مرئيًا أم لا، مع الحفاظ على المساحة التي يشغلها العنصر.

7. تحديد موضع العنصر (Position) 

div {
  position: absolute; /* تحديد موضع العنصر باستخدام إحداثيات */
  top: 10px;
  left: 20px;
}

تستخدم position لتحديد كيفية موضع العنصر. يمكن استخدام قيم مثل absolute و relative و fixed و sticky مع خصائص مثل top و left لتحديد الموقع.

8. التحكم في إخفاء المحتوى الزائد (Overflow) 

div {
  overflow: hidden; /* إخفاء المحتوى الزائد */
}

div {
  overflow: scroll; /* إضافة شريط تمرير للمحتوى الزائد */
}

تستخدم overflow للتحكم في كيفية التعامل مع المحتوى الذي يتجاوز حدود العنصر. يمكن تحديده على أنه hidden أو scroll أو auto.

9. استخدام flexbox (Flex) 

div {
  display: flex; /* تحديد العنصر ليعمل بنظام flexbox */
  justify-content: center; /* محاذاة العناصر أفقيًا في المنتصف */
  align-items: center; /* محاذاة العناصر عموديًا في المنتصف */
}

تستخدم flexbox لتنظيم العناصر داخل الحاوية بطريقة مرنة. باستخدام خصائص مثل justify-content و align-items يمكن محاذاة العناصر بشكل مرن.

10. استخدام grid (CSS Grid) 

div {
  display: grid; /* تحديد العنصر ليعمل بنظام grid */
  grid-template-columns: 1fr 1fr; /* تحديد عمودين بمساواة */
}

تستخدم grid لتقسيم المساحة داخل العنصر إلى صفوف وأعمدة. يمكن تحديد عدد الأعمدة والصفوف باستخدام grid-template-columns و grid-template-rows.

11. تأثيرات التحويل (Transform) 

div {
  transform: rotate(45deg); /* تدوير العنصر بمقدار 45 درجة */
}

تستخدم transform لتطبيق تأثيرات على العنصر مثل التدوير، التحجيم، التحريك، أو التغيير في الشكل.

12. تأثيرات الانتقال (Transition) 

div {
  transition: all 0.3s ease; /* إضافة انتقال تدريجي لجميع الخصائص خلال 0.3 ثانية */
}

تستخدم transition لإضافة تأثيرات انتقالية عندما يتغير العنصر من حالة إلى أخرى.
